# Agent Template

## Rol
Describe la responsabilidad única del agente.

## Entradas mínimas
- Contexto: `INGNOVARTE_CONTEXT.md`
- Objetivo de negocio instruccional
- Restricciones operativas (tiempo, conectividad, licencias)

## Salidas esperadas
- Artefacto markdown corto, verificable y reusable.

## Límites y non-goals
- No crear código ejecutable.
- No prometer integraciones reales.

## Quality Checks
- [ ] Entradas obligatorias completas.
- [ ] Salida trazable a objetivo y evidencia.
- [ ] Límites declarados explícitamente.
