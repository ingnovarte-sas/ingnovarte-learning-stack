# Skill: Learning Analytics Interpreter

## Purpose
Interpretar datos de aprendizaje para soportar decisiones instruccionales.

## When to Use
Cuando ya existen datos de aprendizaje, participación, evaluación o LMS y se requiere lectura accionable.
- Su función es interpretar patrones y convertirlos en hipótesis y acciones.
- No usar para diseñar evaluaciones, rúbricas o instrumentos de campo.

## Inputs
- Datos de evaluación/desempeño.
- Contexto operativo del curso.
- Preguntas de decisión a responder.

## Outputs
- Patrones observados.
- Posibles interpretaciones.
- Hipótesis.
- Alertas.
- Recomendaciones de acción.
- Preguntas pendientes.

## MVP Limits
- No afirmar causalidad si los datos no la soportan.
- No calcula métricas automáticamente; interpreta datos provistos por el usuario.

## Process
1. Validar calidad y contexto de datos.
2. Identificar tendencias y brechas.
3. Separar hallazgos de hipótesis causales.

## Verification Notes
- [ ] Métricas interpretadas con contexto.
- [ ] Brechas y causas probables separadas.
- [ ] Decisiones sugeridas trazables a datos.
