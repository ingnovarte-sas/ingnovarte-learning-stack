# Skill: PPT Storytelling Builder

## Purpose
Diseñar storyboard técnico para presentación PowerPoint enfocada en aprendizaje.

## When to Use
Cuando se necesita guion de diapositivas por objetivo, mensaje y evidencia.
- Usar cuando el entregable sea storyboard, narrativa o guion para PowerPoint.

## MVP Limits
- No diseña la arquitectura completa del curso.

## Inputs
- Objetivos del módulo/sesión.
- Contenido técnico validado.
- Restricciones de formato visual.

## Outputs
Storyboard de diapositivas donde cada slide incluya mínimo: número de slide, mensaje central, evidencia/contenido base, recurso visual sugerido e interacción o decisión del participante cuando aplique.

La narrativa debe construirse por decisiones técnicas y evidencia de operación (problema → criterio técnico → decisión → consecuencia), no como lista plana de temas.

## Process
1. Definir narrativa por decisiones técnicas críticas y evidencia asociada.
2. Ordenar diapositivas por progresión didáctica.
3. Completar campos mínimos verificables por slide.

## Verification Notes
- [ ] Storyline alineado a objetivos.
- [ ] Visuales con intención pedagógica.
- [ ] Cada slide incluye campos mínimos requeridos.
- [ ] Salida compatible con revisión rápida.
