# Skill: Moodle Question Bank Builder

## Purpose
Estructurar banco de preguntas para Moodle en cursos técnicos.

## When to Use
Cuando el entregable es una estructura documental de banco de preguntas para Moodle.
- No usar para diseñar la lógica comparativa completa de prueba inicial/final.
- No usar para evaluar desempeño práctico observable.

## Inputs
- Módulos y objetivos del curso.
- Criterios de evaluación.
- Taxonomía/categorías requeridas.

## Outputs
- Categorías de preguntas.
- Preguntas.
- Opciones.
- Respuesta correcta.
- Retroalimentación.
- Nivel cognitivo.
- Tema o competencia.
- Formato sugerido compatible con carga manual o posterior conversión.

## MVP Limits
- No genera XML Moodle ni integra con Moodle directamente.
- La salida es documental, no ejecutable.

## Process
1. Definir categorías por módulo.
2. Proponer distribución de dificultad.
3. Documentar metadatos mínimos por ítem.

## Verification Notes
- [ ] Categorías coherentes con módulos.
- [ ] Ítems orientados a aplicación práctica.
- [ ] Metadatos mínimos definidos.
