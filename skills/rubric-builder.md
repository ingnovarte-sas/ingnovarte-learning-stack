# Skill: Rubric Builder

## Purpose
Construir rúbricas para evaluar desempeño práctico con criterios observables.

## When to Use
Cuando se requiere evaluar ejecución en terreno con niveles de logro.
- Usar cuando se debe evaluar desempeño observable, producto, práctica o evidencia, no solo respuestas de conocimiento.

## Inputs
- Competencias o tareas críticas.
- Nivel esperado por rol.
- Evidencias observables disponibles.

## Outputs
Rúbrica con dimensiones, niveles y evidencias observables.

## Process
1. Definir dimensiones de desempeño.
2. Redactar niveles con descriptores observables.
3. Verificar aplicabilidad por facilitadores.

## Verification Notes
- [ ] Dimensiones alineadas al trabajo real.
- [ ] Descriptores observables y no ambiguos.
- [ ] Escala aplicable por facilitadores.
