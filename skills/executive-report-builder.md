# Skill: Executive Report Builder

## Purpose
Construir reporte ejecutivo breve de resultados formativos.

## When to Use
Cuando el entregable es una síntesis ejecutiva para toma de decisiones.
- No usar para análisis detallado de datos.
- No usar para diseñar instrumentos de medición o verificación de eficacia.

## Inputs
- Resultados de evaluación/verificación.
- Hallazgos clave y riesgos.
- Audiencia ejecutiva objetivo.

## Outputs
- Resumen ejecutivo.
- Hallazgos clave.
- Implicaciones para decisión.
- Riesgos.
- Recomendaciones ejecutivas.
- Próximos pasos.

## MVP Limits
- No calcula indicadores automáticamente.
- No reemplaza análisis estadístico detallado.

## Process
1. Priorizar hallazgos clave.
2. Sintetizar impacto y límites.
3. Proponer acciones priorizadas.

## Verification Notes
- [ ] Mensaje ejecutivo claro y accionable.
- [ ] Evidencias y límites declarados.
- [ ] Recomendaciones priorizadas.
