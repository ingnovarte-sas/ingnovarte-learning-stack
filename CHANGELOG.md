# Changelog

## v0.1.0 — 2026-04-24

Versión fundacional del **Ingnovarte Learning Stack** como stack markdown-first para diseño instruccional técnico-industrial.

### Resumen
- Creación del stack base markdown-first.
- Definición del `learning-orchestrator` como interfaz principal en lenguaje natural.
- Incorporación de 8 agentes base.
- Incorporación de 12 skills base.
- Incorporación de 7 workflows LDD.
- Definición del protocolo de memoria y uso de memoria curada.
- Incorporación del registry de skills para discovery interno.
- Formalización del contexto reusable de Ingnovarte.
- Inclusión del ejemplo vertical slice de **Hidráulica Nivel II**.
- Consolidación de hallazgos obtenidos durante las pruebas del caso.
- Declaración explícita de límites del MVP.

### Alcance de esta versión
- Stack documental reusable.
- Uso conversacional a través de `learning-orchestrator`.
- Routing interno de agentes, skills y workflows.
- Soporte para diseño, evaluación y verificación de eficacia en contexto técnico-industrial.

### Límites del MVP
- No incluye código ejecutable.
- No incluye CLI.
- No incluye MCP propio.
- No incluye integraciones reales con Moodle, SCORM, Captivate o GitHub.
- No incluye automatización runtime.
